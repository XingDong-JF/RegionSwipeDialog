# PlaceDialog 地区选择组件说明文档

## 组件概述

### 简介

`PlaceDialog` 是一个基于 HarmonyOS ArkUI 开发的**省市区三级联动选择器**组件，支持动态数据加载、初始值设置和结果回调。

### 核心特点

✅ **三级联动** - 省份 → 城市 → 区县自动级联  
✅ **动态显示** - 根据数据层级自动显示 1-3 列  
✅ **初始值支持** - 可预设初始选中项  
✅ **性能优化** - 无防抖设计，响应速度 < 5ms  
✅ **数据校验** - 完整性验证，防止无效选择  

### 适用场景

- 用户地址选择
- 配送范围设置
- 地区筛选功能
- 行政区划查询
